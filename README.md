# Enterprise File Integrity Monitor

A GUI-based cybersecurity monitoring system built using Python and Tkinter.

## Features

- SHA-256 based file integrity verification
- Real-time folder monitoring
- Secure login system
- Automatic ignore system files
- Live activity dashboard
- Change counter tracking

## How to Run

1. Install Python 3.x
2. Run: python main.py
3. Login using system password
4. Select folder
5. Perform initial scan
6. Start monitoring

## Technologies Used

- Python
- Tkinter (GUI)
- hashlib
- threading
- JSON storage
